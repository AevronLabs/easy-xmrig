### To run the script to automatically retrieve the Advanced XMRig folders and files, run the following command: `curl "https://raw.githubusercontent.com/AevronLabs/easy-xmrig/refs/heads/main/advanced-xmrig.sh" -s | /bin/bash`
** To use the Default XMRig instead of the AevronLabs fork, run `export USE_DEFAULT=1` before running the script. **
